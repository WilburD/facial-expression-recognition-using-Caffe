
# How to use ##
(1) copy and paste your face images in database folder.
(2) go to 9_Preprocessing folder, and do the preprocssing as you want. (the data might be saved in database/preprocessed folder.)
(3) go to 1_get_db_list folder and get database list with labels. (The text file might be saved in db_list folder.)
(4) go to 2_create_lmdb folder and get lmdb.
(5) go to 3_Pretrain folder and training the network.
